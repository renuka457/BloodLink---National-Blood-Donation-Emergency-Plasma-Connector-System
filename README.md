BloodLink is a Python-based console application designed to assist in emergency blood and plasma donation by connecting patients with nearby eligible donors based on blood group and city.

## Problem Statement

During medical emergencies, finding compatible blood donors quickly is critical. BloodLink aims to reduce response time by maintaining a searchable donor registry and enabling fast, location-based matching.

## Features

* Add donor details with blood group and city validation
* Search donors by blood group and city
* Suggest nearby donors if exact match is unavailable
* Persistent data storage using JSON files
* Simple and user-friendly menu-driven interface

## Tech Stack

* Programming Language: Python
* Data Storage: JSON
* Concepts Used: File Handling, Lists & Dictionaries, Input Validation

## How It Works

1. Donor information is stored in a JSON file
2. Data loads automatically when the application starts
3. Users can add or search donors through menu options
4. Matching donors are displayed instantly
